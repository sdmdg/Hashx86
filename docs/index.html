> System_Info

Hashx86 is a minimalistic, educational operating system built from scratch. It focuses on core features without unnecessary complexity, serving as a foundation for exploring low-level system programming.

> Execute_Build

user@linux:~/hashx86 
# 1. Clone Repository
$ git clone https://github.com/sdmdg/Hashx86.git
$ cd hashx86

# 2. Setup Disk (Auto-Format on first boot)
$ qemu-img create -f vdi HDD.vdi 1G
$ make run

# 3. Build & Install Drivers
$ cd drivers/ && make
$ cd ../user_prog/ && make
$ cd .. && make hdd

# 4. Launch System
$ make runq
